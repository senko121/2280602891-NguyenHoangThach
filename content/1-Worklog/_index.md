During my internship in the **First Cloud AI Journey (FCAJ)** program, I completed a **12-week internship roadmap** designed to provide comprehensive knowledge and hands-on experience with cloud computing technologies on the AWS platform. The internship progressed from fundamental AWS concepts, infrastructure deployment, system administration, security, and automation to the analysis, design, implementation, testing, and completion of a practical cloud project.

Through theoretical lessons, hands-on laboratory exercises, and professional activities at the AWS office, I progressively developed practical skills in designing, deploying, managing, monitoring, securing, and optimizing cloud infrastructure and applications on AWS.

The weekly activities carried out throughout the internship are summarized as follows:

**Week 1:** [Became familiar with AWS, the working environment, and fundamental IAM concepts, including Users, Groups, Roles, and Policies. Completed Lab02 and Lab03.](1.1-week1/)

**Week 2:** [Built AWS networking infrastructure using VPC, Subnets, and Security Groups; deployed EC2 instances, configured SSH access, deployed a Node.js application, worked with Amazon RDS, and performed database backup and recovery. Completed Lab04 and Lab05.](1.2-week2/)

**Week 3:** [Configured an Application Load Balancer and Auto Scaling Group, managed AWS budgets, monitored cloud costs, and participated in the FCAJ event at the AWS office. Completed Lab06 and Lab07.](1.3-week3/)

**Week 4:** [Implemented a Hybrid DNS architecture using Amazon Route 53 Resolver, monitored infrastructure with Amazon CloudWatch, analyzed system logs, and configured automated monitoring dashboards and alerts. Completed Lab08 and Lab09.](1.4-week4/)

**Week 5:** [Managed cloud infrastructure using AWS CLI, deployed Amazon SNS, built a multi-account environment with AWS Organizations, and configured IAM Identity Center. Completed Lab10, Lab11, and Lab12.](1.5-week5/)

**Week 6:** [Implemented backup and recovery solutions, performed virtual machine migration between on-premises and AWS environments, deployed Docker applications on AWS, and built containerized architecture with Amazon ECS Fargate. Completed Lab13, Lab14, Lab15, and Lab16.](1.6-week6/)

**Week 7:** [Implemented CI/CD pipelines, configured CloudWatch Container Insights, AWS Security Hub, AWS Config, VPC Peering, and AWS Transit Gateway. Completed Lab17, Lab18, Lab19, and Lab20.](1.7-week7/)

**Week 8:** [Researched project ideas, identified business requirements, explored suitable AWS solutions, and proposed the overall implementation approach for the cloud project.](1.8-week8/)

**Week 9:** [Analyzed project requirements, designed the overall system architecture, selected appropriate AWS services, and prepared the project implementation plan.](1.9-week9/)

**Week 10:** [Prepared the deployment environment, integrated system components, evaluated system performance, optimized cloud resources, and prepared the project for the testing phase.](1.10-week10/)

**Week 11:** [Performed comprehensive system testing, optimized performance and resource utilization, validated security, monitoring, backup, and recovery mechanisms, and completed the project's technical documentation.](1.11-week11/)

**Week 12:** [Finalized the AWS cloud project, evaluated the implementation results, completed the technical documentation and final report, presented the project outcomes, incorporated mentor feedback, and successfully completed the internship program.](1.12-week12/)

Throughout the internship, I acquired valuable practical experience in cloud computing, including infrastructure design, application deployment, resource management, system security, deployment automation, and cloud solution development on AWS. This internship has provided a solid foundation for my future career and strengthened my technical expertise in cloud technologies.
